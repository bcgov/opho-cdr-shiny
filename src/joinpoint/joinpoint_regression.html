Building the Model:

Specifics :

  • Model : Linear regression
  • Response (y): Rates
  • Predictors : Sex*Year
  • segmented.lm is used
attach(sample_df)
lm <- lm(STD_RATE_PER_1000~SEX*YEAR)
seg_model<-segmented(lm,seg.Z=~YEAR)
